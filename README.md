# 사용자가 선호하는 책을 통한 영화 추천
책과 영화라는 매체를 합쳐 만든 추천 프로그램

## 💻프로젝트 소개
사용자가 선택한 책의 정보와 넷플릭스에 있는 영화의 줄거리의 유사도를 비교하여 책과 관련 있는 영화를 추천해 준다.

## 🕰️개발 기간
* 2022.04.18 ~ 2022.05.03

## 🧑‍🤝‍🧑프로젝트 참여 인원
* 3명

## 📌주요 기능
* 책 정보 가져오기
* 책 정보 영어로 번역하기
* 영화 줄거리 가져오기
* 책과 영화 유사도 구하기
* 책과 유사한 영화 추천하기

## 🙋 담당 영역
* 책 정보 수집 및 데이터 전처리
* 한글 -> 영어 번역 구현
* TF-IDF & 코사인 유사도 구현
* 영화 추천 구현
